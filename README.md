# ICT-306-Projet-Hacking
Projet de cybersécurité centré sur l’organisation d’un tournoi international de hacking à Lausanne. Développement d’un site web, gestion logistique, sécurité, infrastructure réseau et communication autour de l’événement. Réalisé avec HTML/CSS, Python/Flask et méthodes Agile/SCRUM.
